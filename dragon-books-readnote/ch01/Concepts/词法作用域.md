#  概念：也叫静态作用域

##  定义
> 变量的作用域在**编译阶段**（书写代码时）就确定了。函数在定义它的环境中寻找变量，而不是在调用它的环境中寻找。

##  笔记 / 例子
``` python
x = 10

def func_A():
    print(x)

def func_B():
    x = 20
    func_A()

func_B() 
# 输出结果：10
```
``` csharp
using System;

class Program {
    static int x = 10; // 全局（类级别）变量

    static void FuncA() {
        Console.WriteLine(x); // 查找环境：FuncA定义在类中，它能看到类级别的 x
    }

    static void FuncB() {
        int x = 20; // 局部变量，属于 FuncB 的环境
        FuncA();    // 调用 FuncA
    }

    static void Main() {
        FuncB();
    }
}
```
